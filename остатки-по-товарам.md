# Остатки по товарам — База знаний Aqua Delivery

Если вам необходимо включить отслеживание остатков по товару, обратитесь в техническую поддержку для подключения данного функционала.  
После этого у вас появится в кабинете настроек в левом боковом меню раздел **«****Остатки по товарам»** и вы можете настроить отслеживание остатка товаров в приложении. При заказе товара через приложение, система вычтет количество купленного товара из остатков.

Для того, чтобы включить отслеживание остатков по товарам, войдите в **Кабинет настроек**и в боковом меню выберите раздел **«****Остатки по товарам»**.

![](https://howto.aqua-delivery.com/wp-content/uploads/2020/12/Screenshot_1.png)

На этой странице отображены все товары, имеющиеся в системе, и остаток для каждого товара.  
![](https://howto.aqua-delivery.com/wp-content/uploads/2020/12/Screenshot_2.png)

Для того, чтобы указать остаток для конкретного товара, нажмите на кнопку в столбце **«****Пополнить товар»** (кружок со стрелочкой) в строке с нужным товаром и укажите количество товара в остатках.

![](https://howto.aqua-delivery.com/wp-content/uploads/2020/12/Screenshot_3-1.png)  
![](https://aqua-delivery.ru/howto/wp-content/uploads/2020/12/Screenshot_15-700x302.png)

Нажмите на кнопку **«****Пополнить»**.

Если хотите уменьшить количество остатка по товару, необходимо в строке прописать число с минусом, например, в наличии ранее было 50 бутылей, теперь 45. В таком случае в строке нужно написать «-5» и нажать **«****Пополнить»**. 

![](https://howto.aqua-delivery.com/wp-content/uploads/2020/12/Screenshot_4-2.png)

Если у вас сейчас нет товара в наличии и вы хотите проинформировать клиентов о том, что он будет позднее, вам необходимо прописать то же самое число, которое у вас стоит только с минусом и нажать **«****Пополнить»**. В таком случае в мобильном приложении возле данного товара появится надпись **«****Будет позже».**

![](https://howto.aqua-delivery.com/wp-content/uploads/2020/12/Screenshot_7.png)

![](https://howto.aqua-delivery.com/wp-content/uploads/2020/12/Screenshot_11-2.png)

![](https://howto.aqua-delivery.com/wp-content/uploads/2020/12/photo_2025-03-20_11-28-33.jpg)

Чтобы снова сделать количество товара бесконечным, нажмите на кнопку в последнем столбце **«****Сделать бесконечным».**  
![](https://howto.aqua-delivery.com/wp-content/uploads/2020/12/Screenshot_5-1.png)

Далее выйдет окошко с подтверждением, необходимо нажать **«Ок»**.

![](https://howto.aqua-delivery.com/wp-content/uploads/2020/12/Screenshot_6-1.png)
