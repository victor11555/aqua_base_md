# Как настраивать работу с КИГУ — База знаний Aqua Delivery

Перед началом работы по настройке КИГУ нужно обратить внимание на важный момент: в монолите заводится два товара (штучный и упаковка), а в 1С только один.

Перед настройкой упаковки необходимо добавить нужные единицы измерения в справочник.  
Перейдите в раздел: «Номенклатура» → «Товар» → «Единицы измерения». Там укажите единицу измерения «Упаковка» и настройте количество товара в упаковке (коэффициент).![](https://howto.aqua-delivery.com/wp-content/uploads/2025/07/1-1.png)Так как упаковка товара и штучный товар имеют разные GTIN, оба эти идентификатора нужно прописать в 1С в разделе «Штрихкоды».![](https://howto.aqua-delivery.com/wp-content/uploads/2025/07/2-1.png)После этого нужно произвести сопоставление товаров между монолитом и 1С. Для этого переходим: «CRM» → «Интернет-магазин» → «Обслуживание» → «Инструменты администратора» → «Соответствия объектов» → «Создать».![](https://howto.aqua-delivery.com/wp-content/uploads/2025/07/3-1.png)Далее подтянется форма настройки соответствия объектов. В ней заполняем:

- Идентификатор магазина и идентификатор базы — ID товара из монолита;
- Ссылка на объект — выбираем в Единицах измерения нужную упаковку (например, упаковка 12шт или упаковка 6шт);
- Ставим признак «Объект выгружен»;
- Тип объекта — Справочники. Единицы Измерения;
- Записываем и закрываем форму с изменениями.
