# Сопоставление курьеров из Вод@воз — База знаний Aqua Delivery

Для сопоставления курьеров из Вод@воз необходимо перейти в кабинет настроек, затем выбрать пункт «Настройки».

![](https://howto.aqua-delivery.com/wp-content/uploads/2023/10/Рисунок1.png)

Затем выбрать пункт «Обмен» и в нем «Курьеры».

![](https://howto.aqua-delivery.com/wp-content/uploads/2023/10/photo_2026-01-21_13-50-14.jpg)

У вас откроется окно «Водители», в котором необходимо нажать на кнопку «Создать».

![](https://howto.aqua-delivery.com/wp-content/uploads/2023/10/Рисунок3.png)

Далее требуется выбрать курьера из нашего сервиса и ввести логин/пароль от Вод@воз.

![](https://howto.aqua-delivery.com/wp-content/uploads/2023/10/Рисунок4.png)

Таким образом вам удастся сопоставить курьеров из Вод@воз с нашим сервисом.
