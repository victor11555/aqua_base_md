# Добавление адреса в мобильном приложении — База знаний Aqua Delivery

Для того, чтобы добавить адрес в мобильном приложении:

1. Зарегистрируйтесь или войдите в аккаунт, если вы уже зарегистрированы.
2. На главном экране приложения сверху нажмите на выпадающий список рядом с адресом.

![](https://howto.aqua-delivery.com/wp-content/uploads/2020/08/photo_2025-03-13_22-20-25.jpg)

3. Вам откроется список сохраненных адресов. 

На этой странице вы можете:

- **Выбрать другой адрес**. Выбранный адрес обозначается синей галочкой.
- **Удалить адрес**. Для этого нужно нажать на кнопку **«****Корзина»** и подтвердить удаление.
- **Добавить адрес**. Нужно нажать на кнопку + и внести новый адрес.

![](https://howto.aqua-delivery.com/wp-content/uploads/2020/08/photo_2025-03-13_22-23-15.jpg)

Адрес доставки может быть добавлен в приложение следующими способами:

- **Определен автоматически**. Если приложению был разрешен доступ к определению местоположения.
- **Внесен вручную.** Адрес в строке можно изменить вручную, для этого нужно внести город, улицу, дом.
- **Показан на карте**. Для этого нужно навести курсор (фото ниже) на нужный дом, адрес будет определен автоматически.

![](https://howto.aqua-delivery.com/wp-content/uploads/2020/08/photo_2025-03-13_20-55-44.jpg)

4. Также вы можете добавить адрес, когда приступили к оформлению заказа. Необходимо нажать «Указать другой адрес» и в выпадающем списке либо выбрать существующий адрес, либо добавить новый.

![](https://howto.aqua-delivery.com/wp-content/uploads/2020/08/photo_2025-03-13_22-30-43.jpg)
