# Инструмент сокращения ссылок — База знаний Aqua Delivery

Инструмент разработан на базе системы Aqua Delivery для упрощения работы с длинными ссылками — https://short.appsol.ru

Как им пользоваться:

1. Откройте инструмент перейдя по ссылке (авторизация никакая не требуется) — https://short.appsol.ru
2. Вставьте в первое поле ссылку, которую хотите сократить  
   ![](https://howto.aqua-delivery.com/wp-content/uploads/2022/10/1.png)
3. Если вы хотите, чтобы после косой черты было написано что-то уникальное, то введите это используя латинские буквы и цифры, а если не надо, то можно оставить пустым, система сама сгенерирует набор букв и цифр (можно не заполнять это поле, например, если ссылка нужна для QR-кода). Одно уникальное обозначение можно использовать только один раз, если кто-то до вас его уже использовал, то применить его не получится.  
   ![](https://howto.aqua-delivery.com/wp-content/uploads/2022/10/Пиши-сокращай-Google-Chrome-2022-10-17-14.57.3.png)
4. Нажмите на сократить. Ниже появится готовая ссылка, которую можно применять в работе (после нажатия на кнопку «Сократить», сокращенная ссылка автоматически копируется в буфер обмена)  
   ![](https://howto.aqua-delivery.com/wp-content/uploads/2022/10/image_2022-10-17_15-04-52.png)
