# Возврат денежных средств при онлайн-оплате — База знаний Aqua Delivery

1. Перейдите в кабинет заказов и нажмите на три точки в верхней панели управления, затем выберите раздел **Онлайн оплата.**![](https://howto.aqua-delivery.com/wp-content/uploads/2023/06/2024-05-28_14-30-16.png)
2. Кнопка «Возврат оплаты» доступна пользователям с правами администратор! Для пользователя с ролью оператор данный функционал недоступен. Найдите заказ, у которого нужно провести возврат и нажмите на кнопку возврата. **Важно:**если в поле СБП стоит галочка, то возврат средств на данном шаге окончен.**** ****![](https://howto.aqua-delivery.com/wp-content/uploads/2023/06/2024-05-28_14-32-35.png)****
3. Если у вас подключена интеграция с OFD.ру, то после совершения возврата автоматически формируется чек возврата, который можно будет посмотреть в заказе, в разделе чеки. **Важно: если оплата была через эквайринг, то возврат средств нужно произвести через личный кабинет банка.  
   ![](https://howto.aqua-delivery.com/wp-content/uploads/2023/06/2023-06-30_16-22-11.png)**

- **Если у вас Сбербанк, то перейдите в личный кабинет по ссылке.**
- Перейдите в раздел **«Транзакции»,** найдите нужный заказ для возврата и перейдите в него (**Ищите по сумме заказа и времени отправки и если есть по почте**).  
  ![](https://howto.aqua-delivery.com/wp-content/uploads/2023/06/photo_5258289771883646006_y.jpg)
- Нажмите кнопку **«Возврат»** в правом верхнем углу.![](https://howto.aqua-delivery.com/wp-content/uploads/2023/06/photo_5258289771883646007_y.jpg)
