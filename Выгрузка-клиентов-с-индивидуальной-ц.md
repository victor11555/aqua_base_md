# Выгрузка клиентов с индивидуальной ценой — База знаний Aqua Delivery

Для экспорта данных о клиентах с установленными индивидуальными ценами необходимо выполнить следующие действия:

1. Перейдите в кабинет заказов.
2. Откройте раздел **«Клиенты»**, там вы увидите полный список ваших клиентов.
3. На открывшейся странице, в левом верхнем углу, найдите и нажмите кнопку **«Скачать»**.
4. В выпадающем меню выберите опцию **«Выгрузить клиентов с индивидуальными ценами»**.

После этого система автоматически сформирует и загрузит на ваше устройство файл в формате **Excel**, содержащий информацию обо всех клиентах с настроенными персональными ценами.
