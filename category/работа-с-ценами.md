# Работа с ценами — База знаний Aqua Delivery

Работа с ценами

Для того, чтобы добавить типы цен, выполните следующие действия: 1. Войдите в Кабинет настроек (admin.доменное\_имя\_вашей\_компании.aqua-delivery.ru). 2. В боковом меню нажмите на пункт «Справочники» и выберите раздел «Типы цен». 3. Откроется страница «Типов цен». 4. На этой странице вы можете редактировать приоритет цены товара. Эта функция нужна для того, чтобы тип цены корректно отображался в приложении. […]

Работа с ценами

Настройка цен Для того, чтобы настроить цену на товар в зависимости от количества, войдите в Кабинет настроек (admin.домен\_вашей\_компании.aqua-delivery.ru), в боковом меню нажмите на пункт «Товары» и выберите категорию товара. После этого у вас откроется страница с товарами в категории. Нажмите на значок с долларом напротив товара, цену которого вы хотите отредактировать. Откроется окно редактирования цен. […]
