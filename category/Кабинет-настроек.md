# Кабинет настроек — База знаний Aqua Delivery

Кабинет настроек

Для того чтобы отправлять пуш-уведомления клиентам с определенной зоны доставки, необходимо зайти в Кабинет настроек. Далее перейти в раздел «Клиенты». Выполните экспорт в CSV. Откройте скачанный файл. Оставьте только поля ID и зона доставки, остальные удалите. Выделите столбец «Зона». Нажмите «Сортировка и фильтр» и выберите нужную зону. Перейдите в Кабинете настроек в раздел «Создать пуш». […]
