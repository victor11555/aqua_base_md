# Без рубрики — База знаний Aqua Delivery

Без рубрики

Шаги: Скачайте экземпляр Excel-файла или сопоставьте ваш Excel файл так, чтобы порядок столбцов совпадал с примером из экземпляра. Передайте файл для импорта на проверку специалистам технической поддержки AquaDelivery. В случае возникновения вопросов специалисты отправят вам документ на доработку.
