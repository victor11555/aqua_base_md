# Раздел «Планировщик заказов» — База знаний Aqua Delivery

**Планировщик заказов —** удобная функция, поскольку с её помощью есть возможность создавать заказы автоматически. Для того чтобы найти данный раздел, в **Кабинете заказов** необходимо нажать на три точки.

![](https://howto.aqua-delivery.com/wp-content/uploads/2022/05/1.png)

При переходе в раздел открывается весь необходимый функционал и заказы, которые уже запланировали ваши операторы.

![](https://howto.aqua-delivery.com/wp-content/uploads/2022/05/1-1.png)

Среди уже запланированных заказов оператор может найти конкретный заказ по номеру, имени клиента, адресу, дню / времени доставки и дате создания. Также имеется возможность увидеть состав запланированных заказов.

![](https://howto.aqua-delivery.com/wp-content/uploads/2022/05/1-2.png)

Для создания заказа-шаблона необходимо нажать на кнопку **«+ Новый заказ-шаблон»**.

![](https://howto.aqua-delivery.com/wp-content/uploads/2022/05/1-3.png)

Затем выйдет окно, в котором нужно выбрать подходящие номер заказа-шаблона, день доставки и интервал. Не забываем сохранить.

![](https://howto.aqua-delivery.com/wp-content/uploads/2022/05/1-4.png)

Обращаем ваше внимание на то, что создание заказа происходит за 48 часов до начала интервала.

При этом заказ-шаблон должен соответствовать следующим критериям:

1. Он должен быть привязан к зарегистрированному пользователю.
2. Иметь конкретный адрес доставки (не самовывоз).

Тип оплаты может быть любым, кроме Онлайн-оплаты и Оплаты баллами.
