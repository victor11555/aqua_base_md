# Поиск по коду маркировки — База знаний Aqua Delivery

Для поиска КМ необходимо отсканировать его через приложение Честный Знак. Бизнес:  
1. Откройте приложение и перейдите в сканер, наведите сканер на КМ и нажмите «Проверить код»;

![](https://howto.aqua-delivery.com/wp-content/uploads/2025/10/photo_2025-10-01_14-14-55.jpg)

2. Перед Вами откроется результат сканирования со всеми данными товара. Для того, чтобы найти код маркировки, нажмите на кнопку «Посмотреть структуру»;

![](https://howto.aqua-delivery.com/wp-content/uploads/2025/10/photo_2025-10-01_14-14-56.jpg)

3. Откроется страница «Описание КМ» и из графы «Штрихкод» нужно скопировать данный код маркировки.

![](https://howto.aqua-delivery.com/wp-content/uploads/2025/10/2025-10-10_11-35-05.png)

Для того, чтобы воспользоваться функцией «Поиск по коду маркировки», необходимо совершить следующие действия:  
1. Переходим в кабинет настроек;  
2. Развернуть вкладку «Настройки»;  
3. Открыть раздел «Поиск по коду маркировки»;

![](https://howto.aqua-delivery.com/wp-content/uploads/2025/10/photo_2025-09-24_13-45-48.jpg)

4. В строке «Введите код маркировки для поиска заказа» необходимо прописать код, отсканированный с бутылей и нажмите кнопку «Поиск». В строке нужно прописать только криптохвост или указать значения без криптохвоста, как прописано на скриншоте.

![](https://howto.aqua-delivery.com/wp-content/uploads/2025/10/2025-10-10_11-37-32.png)

5. Появится ссылка с переходом на заказ, в котором был отсканирован данный код.

![](https://howto.aqua-delivery.com/wp-content/uploads/2025/10/photo_2025-09-03_16-38-48.jpg)

Обращаем Ваше внимание на то, что данная функция работает и по поиску КИТУ.
