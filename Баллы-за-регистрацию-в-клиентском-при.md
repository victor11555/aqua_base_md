# Баллы за регистрацию в клиентском приложении — База знаний Aqua Delivery

Для того, чтобы клиенту начислялись баллы за регистрацию, необходимо:  
1. Зайти в кабинет настроек;  
2. Развернуть вкладку «Система лояльности»;  
3. Перейти в раздел «Настройки»;

![](https://howto.aqua-delivery.com/wp-content/uploads/2025/10/photo_2025-10-02_16-55-52.jpg)

Перед вами откроется страница с настройками «Системы лояльности»:

![](https://howto.aqua-delivery.com/wp-content/uploads/2025/10/photo_2025-10-08_11-56-28.jpg)

Далее:  
4. Прокрутить страницу вниз до конца;  
5. В разделе «Регистрация» в строку «Баллов за регистрацию» указать нужное количество баллов;  
6. Нажимаем кнопку «Сохранить».

![](https://howto.aqua-delivery.com/wp-content/uploads/2025/10/photo_2025-10-08_11-55-00.jpg)
