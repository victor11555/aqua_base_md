# Отчет по трекерам — База знаний Aqua Delivery

Перейти к просмотру отчета по конкретному трекеру можно из двух мест:

- со страницы со списком трекеров (нажать на “Отчет”);  
  ![](https://howto.aqua-delivery.com/wp-content/uploads/2024/01/отчет1.png)
- В нужном отчете через фильтр (в отличие от предыдущего варианта нужно фильтр настроить вручную). Для настройки фильтра выберите: Сегментировать по пользователям, у которых — Установки — Трекер — Включить — *Выберите нужные пункты.*  
  ![](https://howto.aqua-delivery.com/wp-content/uploads/2024/01/отчет2.png)

Основной отчет, который будет наиболее полезен для отслеживания эффективности — установки (User Acquisition).  
![](https://howto.aqua-delivery.com/wp-content/uploads/2024/01/отчет3.png)

Помимо основных фильтров сверху, есть возможность выводить определенные метрики ниже графика. Для этого нужно нажать на “+ Группировки и метрики”.  
![](https://howto.aqua-delivery.com/wp-content/uploads/2024/01/отчет4.png)

**Группировки** — это строки. В основном в качестве группировок выступают периоды времени.

**Метрики** — это столбцы. В столбцах обычно выводятся необходимые параметры с фактическими значениями.

Чтобы добавить группировки или метрики, необходимо нажать на “+”.  
![](https://howto.aqua-delivery.com/wp-content/uploads/2024/01/отчет5.png)

Наиболее полезные метрики, которые стоит брать в расчеты:

- Клики — количество переходов по ссылке;
- Установки — количество установивших приложение (всегда меньше чем кликов);
- Сессии — количество запусков приложений (сколько раз его открывали);
- Покупки — количество событий “Сделали заказ”;
- Доход — сумма денег, на которую сделали заказов;
- Пользователи, купившие товар — количество активных пользователей, которые сделали заказ;
- Средний доход на пользователя (ARPU) — доход / пользователи, купившие товар;
- Покупок на пользователя — покупки / пользователи, купившие товар.  
  ![](https://howto.aqua-delivery.com/wp-content/uploads/2024/01/отчет6.png)
