# Типы товаров — База знаний Aqua Delivery

**В системе Aqua Delivery предусмотрены следующие типы товаров:**

1. Вода 19 л — товары, работающие с возвратной тарой.
2. Оборудование —  дополнительные товары (кулеры, помпы и т.п.)
3. Напитки — дополнительные товары (включая воду объёмом меньше 19 л).

Типы товаров устанавливаются в настройках товара через кабинет настроек.  
![](https://howto.aqua-delivery.com/wp-content/uploads/2022/09/2025-03-21_15-14-56.png)

В кабинете заказов при наведении на специальные иконки отображается краткий состав заказа.  
![](https://howto.aqua-delivery.com/wp-content/uploads/2022/09/2025-03-21_15-18-15.png)

1. Первая иконка — вода 19 л., количество товаров с типом товара «Вода 19 л».
2. Вторая иконка — дополнительные товары, количество товаров с типами товаров «Напитки» и «Оборудование».

**Для типов товаров можно применить следующие настройки:**

1. Минимальное количество товара с определенным типом в заказе.  
   В кабинете настроек необходимо перейти в раздел «Настройки — Справочники — Типы товаров».  
   ![](https://howto.aqua-delivery.com/wp-content/uploads/2022/09/2025-03-21_15-39-51.png)
2. Далее в столбце «Минимальное количество в заказе» нажимаем на цифру, относящуюся к ограничению по количеству товара, прописываем нужное значение и сохраняем изменения.  
   ![](https://howto.aqua-delivery.com/wp-content/uploads/2022/09/2025-03-21_16-17-37.png)Когда клиент попытается оформить в приложении заказ с меньшим количеством товара, то он увидит сообщение о минимально допустимом количестве товаров данного типа и не сможет оформить заказ:  
   ![](https://howto.aqua-delivery.com/wp-content/uploads/2022/09/IMG_2196.png)
3. Платная доставка при определенном количестве для типов товаров.  
   В кабинете настроек необходимо перейти в раздел «Настройки — Справочники — Стоимость доставки».  
   ![](https://howto.aqua-delivery.com/wp-content/uploads/2022/09/2025-03-21_15-57-29.png)
4. Далее создаём сценарий, при котором будет действовать стоимость доставки. Нажимаем кнопку «Создать», переходим в раздел, где нужно прописать всю информацию — от какого количества и для каких типов товаров будет действовать данная настройка, а также саму стоимость доставки.![](https://howto.aqua-delivery.com/wp-content/uploads/2022/09/2025-03-21_16-09-21.png)![](https://howto.aqua-delivery.com/wp-content/uploads/2022/09/2025-03-21_16-05-39.png)  С дальнейшими действиями по редактированию определённого сценария вы можете ознакомиться в пункте 2 данной инструкции.
