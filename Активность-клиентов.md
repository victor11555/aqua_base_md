# Активность клиентов — База знаний Aqua Delivery

Для просмотра активности клиентов нужно выполнить следующие действия:

1. Войдите в **Кабинет настроек.**
2. В боковом меню выбрать раздел **Активность клиентов.**

![](https://howto.aqua-delivery.com/wp-content/uploads/2022/06/Screenshot_1-1.png)

3. Откроется раздел **Активности клиентов.**

Раздел содержит информацию о пользователях приложения и устройствах, с которых осуществлялся вход в приложение.

![](https://howto.aqua-delivery.com/wp-content/uploads/2022/06/Screenshot_5-2.png)

Например, пользователь Виктория установила приложение и авторизовалась в нем. Статус «Не авторизован» обозначает пользователей, которые установили приложение, но не вошли в свой аккаунт.

![](https://howto.aqua-delivery.com/wp-content/uploads/2022/06/Screenshot_7-1.png)

4. Страница Активности клиентов содержит следующую информацию:

- Пользователь.
- Авторизация. Показывает авторизован клиент в приложении или нет.
- Активность. Статус клиента в карточке, в кабинете настроек (может быть статус «Удален», тогда клиент не сможет войти в приложение).
- ID устройства.
- Дата установки приложения. На определенное устройство.
- Дата удаления приложения. Если клиент не заходил в приложение больше 30 дней, система считает, что он удалил приложение.
- Дата последнего использования приложения.
- Платформа устройства.
- Версия системы устройства.
- Версия приложения.
