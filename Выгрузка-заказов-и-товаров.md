# Выгрузка заказов и товаров — База знаний Aqua Delivery

Инструменты для экспорта данных находятся на главной странице кабинета заказов. Для этого выберите в левом верхнем углу кнопку «Скачать».

![](https://howto.aqua-delivery.com/wp-content/uploads/2022/05/Screenshot_9-4.png)

Если вы хотите сделать выгрузку большого объёма заказов (более 10 тыс. строк), то вам необходимо сделать экспорт заказов в CSV (примерное время выгрузки: до 3-х минут 10 тыс. строк).

**Виды экспорта:**

1. **Экспорт заказов в csv.** Выгрузка данных о заказах.

![](https://howto.aqua-delivery.com/wp-content/uploads/2022/05/photo_2024-01-09_12-37-26.jpg)

Есть возможность выбрать определенные поля для загрузки: Все поля, ID заказа, ID пользователя, ФИО пользователя, E-mail пользователя, Тип (физлицо, организация), Телефон, Комментарий оператора к клиенту, Адрес, Комментарий к адресу, Зона доставки, Район доставки, Состав, Кол-во возвращенной тары, Комментарий клиента к заказу, Комментарий оператора к заказу, Водитель, Статус заказа, Сумма заказа, Сумма оплаты, Сумма платной доставки, Кешбэк за заказ, Принято наличных, Способ оплаты, Статус оплаты, Первый заказ, Интервал доставки (время), Интервал доставки (дата), Дата и время доставки, Оценка курьера, Комментарий к оценке курьера, Дата создания заказа, Источник заказа, ИНН.

Вам необходимо выбрать нужные поля для выгрузки и нажать на кнопку «Скачать».

![](https://howto.aqua-delivery.com/wp-content/uploads/2022/05/photo_2024-01-09_12-37-34.jpg)

![](https://howto.aqua-delivery.com/wp-content/uploads/2022/05/Screenshot_5-4.png)

**2. Экспорт товаров в csv.** Выгрузка данных о товарах.

![](https://howto.aqua-delivery.com/wp-content/uploads/2022/05/photo_2024-01-09_12-39-30.jpg)

Есть возможность сортировки выгрузки товаров по полям: Все поля, ID заказа, ID пользователя, ФИО, ID товара, Название товара, Кол-во товара, Цена за единицу, Общая цена, Способ оплаты, Источник заказа. После выбора нужных полей нажмите «Скачать». 

![](https://howto.aqua-delivery.com/wp-content/uploads/2022/05/photo_2024-01-09_12-40-03.jpg)
