# Appmetrica — База знаний Aqua Delivery

**Трекинговая система (трекер)** – это, в первую очередь, инструмент для отслеживания источников трафика приложения. Трекинговая система необходима для того, чтобы отслеживать из каких источников были переходы, установки и их количество. В системе Aqua Delivery трекер реализован через сайт Appmetrica — сервис для трекинга и аналитики мобильных приложений.

В Appmetrica на данный момент реализованы следующие возможности:

- Просмотр количества новых и постоянных пользователей.
- Мониторинг способов установки.
- Просмотр статистики удержания пользователей.
- Просмотр соотношения клиентов скачавших приложение и дошедших до оформления заказа.
- Просмотр количества кликов и переходов по рекламе.
- Просмотр статистики покупок.

Чтобы получить доступ к Appmetrica обратитесь в службу технической поддержки Aqua Delivery. Вам предоставят доступ для просмотра информации о вашей компании.

При переходе в Appmetrica  вы попадете в следующие окно:![](https://howto.aqua-delivery.com/wp-content/uploads/2023/06/2023-06-26_17-08-30.png)

Перейдем к рассмотрению отдельно каждого виджета(Окно содержащие информацию о определенном событии).

**Виджет «Установки»**

В данном виджете мы можем посмотреть из какого источника больше всего клиенты скачивают приложение.

**Органика** — клиенты нашли ссылку на скачивание приложения через браузер.

**Основной**— рекламная кампания.

![](https://howto.aqua-delivery.com/wp-content/uploads/2023/06/2023-06-26_17-10-20.png)

**Виджет «Retentio»**

В данном виджете показывается статистика удержания пользователей (пользователи вернувшиеся в ваше приложение спустя время).

![](https://howto.aqua-delivery.com/wp-content/uploads/2023/06/2023-06-26_17-30-03.png)

**Виджет «Конверсия»**

Конверсия показывает процент клиентов дошедших до оформления заказа от общего числа клиентов скачавших приложение.

![](https://howto.aqua-delivery.com/wp-content/uploads/2023/06/2023-06-26_17-32-19.png)

**Виджет «Аудитория»**

В данном графике показано количество клиентов открывших ваше приложение в определенный день.

![](https://howto.aqua-delivery.com/wp-content/uploads/2023/06/2023-06-26_17-35-04.png)

Если кликнуть по заголовку «Аудитория», то вы перейдете в новое окно, в котором можно узнать количество новых клиентов.

Для это нажмите на выпадающий список в правом верхнем углу и оставьте галочку только у пункта «Новые пользователи» и нажмите кнопку «Применить».

![](https://howto.aqua-delivery.com/wp-content/uploads/2023/06/Screenshot_1-1.png)

**Виджет «Клики»**

Показывает переходы по вашим рекламам. Если хотите запустить новую рекламную кампанию обратитесь в службу технической поддержки Aqua Delivery.  Сотрудник поддержки предоставит вам ссылку с UTM-метками, которую можно будет отслеживать в данном графике и графике «Установки»![](https://howto.aqua-delivery.com/wp-content/uploads/2023/06/2023-06-26_17-41-43.png)

**Виджет «Ecom покупки»**

В данном виджете можно отслеживать количество сделанных продаж.![](https://howto.aqua-delivery.com/wp-content/uploads/2023/06/2023-06-26_17-47-40.png)
