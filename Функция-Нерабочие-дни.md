# Функция «Нерабочие дни» — База знаний Aqua Delivery

Для того, чтобы настроить нерабочие дни, выполните следующие шаги:

1. Перейдите в кабинет настроек (admin.«доменное\_имя\_вашей\_компании».aqua-delivery.ru).  
2. Откройте раздел Настройки — Нерабочие дни.

![](https://howto.aqua-delivery.com/wp-content/uploads/2022/05/Screenshot_1.png)

3. В верхнем левом углу нажмите на кнопку «Создать».

![](https://howto.aqua-delivery.com/wp-content/uploads/2022/05/Screenshot_2.png)

4. Выберите дату и время начала действия ограничения в поле «От».

![](https://howto.aqua-delivery.com/wp-content/uploads/2022/05/Screenshot_7-5.png)

5. Выберите дату окончания действия нерабочего дня в поле «До».

![](https://howto.aqua-delivery.com/wp-content/uploads/2022/05/Screenshot_11-4.png)

6. Затем задайте название для данного ограничения в поле «Комментарий». ![](https://howto.aqua-delivery.com/wp-content/uploads/2022/05/Screenshot_12-2.png)

![](https://howto.aqua-delivery.com/wp-content/uploads/2022/05/Screenshot_13-1.png)

7. Сохраните внесенные изменения.

Когда клиент будет оформлять заказ на день, который относится к нерабочему, приложение известит его об ограничении.

![](https://howto.aqua-delivery.com/wp-content/uploads/2022/05/photo_2025-03-16_20-12-09.jpg)
