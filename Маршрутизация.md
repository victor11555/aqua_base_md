# Маршрутизация — База знаний Aqua Delivery

В нашей системе существует функция маршрутизации, которая позволяет курьерскому приложению автоматически выстраивать маршрут для водителя на основе одного интервала. Принцип работы:

1. Оператор назначает на заказы водителя в кабинете заказов. Подробнее о назначении водителя вы узнаете в нашей инструкции: https://howto.aqua-delivery.com/Как-назначить-водителя-на-заказ/.

![](https://howto.aqua-delivery.com/wp-content/uploads/2022/05/photo_2026-01-23_14-18-54.jpg)

2. В курьерском приложении у водителя появится список всех заказов на конкретный интервал доставки.

![](https://howto.aqua-delivery.com/wp-content/uploads/2022/05/photo_2025-03-21_14-21-56.jpg)

3. Чтобы построить маршрут на определенный интервал доставки, выберите интервал, нажав на него. На открывшейся странице вы увидите все заказы на заданный интервал. Курьер сможет менять заказы местами, выстраивая оптимальный маршрут для себя. Затем необходимо выбрать «Построить маршрут».

![](https://howto.aqua-delivery.com/wp-content/uploads/2022/05/photo_2025-03-21_14-21-57.jpg)

4. После этого приложение на карте построит маршрут курьеру по адресам.

![](https://howto.aqua-delivery.com/wp-content/uploads/2022/05/photo_2025-03-21_14-21-58.jpg)
