# Создание и настройка номенклатуры — База знаний Aqua Delivery

**Номенклатура** — перечень товаров и услуг, которые предоставляются компанией, то есть ассортимент.

Сначала товар добавляется в кабинете настроек (см. инструкцию «Добавление нового товара«).

Для создания номенклатуры в 1С следует выполнить действия по алгоритму:

1. Перейдите в раздел «Склад — Номенклатура». В данном разделе отображается вся настроенная номенклатура.![](https://howto.aqua-delivery.com/wp-content/uploads/2025/07/1-8.png)
2. Для создания новой номенклатуры выберите «Создать».![](https://howto.aqua-delivery.com/wp-content/uploads/2025/07/2-8.png)
3. В открывшейся форме заполните все необходимые поля. Важно обратить внимание: поле «Наименование» — то, как в вашей системе будет называться товар; поле «Наименование для печати» — то, как в печатных формах документов будет отображаться номенклатура.  
   Габариты и вес прописываются только для воды 19 л. Как правило, габариты воды 19 л указываются как 100×100×100.![](https://howto.aqua-delivery.com/wp-content/uploads/2025/07/3-7.png)
4. ВАЖНО: Для продукции с маркировкой обязательно указываем «Упакованная вода» и прописываем GTIN товара в разделе «Штрихкоды».![](https://howto.aqua-delivery.com/wp-content/uploads/2025/07/4-6.png) ![](https://howto.aqua-delivery.com/wp-content/uploads/2025/07/5-5.png)

**После создания номенклатуры в 1С, нужно сопоставить товары с соответствующими позициями из кабинета настроек.** Для этого следует:

1. Перейдите в раздел «CRM» → «Интернет магазин».![](https://howto.aqua-delivery.com/wp-content/uploads/2025/07/6-5.png)
2. Переходим в «Обслуживание» → «Соответствия объектов».![](https://howto.aqua-delivery.com/wp-content/uploads/2025/07/7-5.png)
3. Приступаем к созданию соответствия, нажимаем «Создать».![](https://howto.aqua-delivery.com/wp-content/uploads/2025/07/8-5.png)
4. В подтянувшейся форме заполните:
   - идентификатор магазина, идентификатор базы — это ID товара из кабинета настроек;![](https://howto.aqua-delivery.com/wp-content/uploads/2025/07/9-4.png)![](https://howto.aqua-delivery.com/wp-content/uploads/2025/07/10-2.png)
   - ссылка на объект — выбираем «Номенклатура» и товар, который сопоставляем;![](https://howto.aqua-delivery.com/wp-content/uploads/2025/07/11-2.png)
   - проставляем галочку «Объект выгружен»;
   - тип объекта — прописываем «Справочник. Номенклатура».

Полностью заполненная форма должна выглядеть следующим образом:![](https://howto.aqua-delivery.com/wp-content/uploads/2025/07/12-2.png)

Нажимаем «Записать и закрыть». После этого запись о соответствии появится в списке. ![](https://howto.aqua-delivery.com/wp-content/uploads/2025/07/13-2.png)
