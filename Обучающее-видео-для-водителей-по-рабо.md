# Обучающее видео для водителей по работе в курьерском приложении Aqua Delivery — База знаний Aqua Delivery

Рассказываем, как в приложении для водителей изменить тип оплаты и количество товара, построить маршрут до покупателя и посмотреть отчет по количеству заказов, бутылей и принятых денежных средств по способам оплаты. Вы узнаете, как работать с маркировкой и отправлять клиенту push-уведомление из приложения. Найдете информацию об основных разделах: Заказы, Архив, Чат, Профиль.
