### Создание нового языка

В системе управления можно заводить разные языки для перевода.

1. Для этого необходимо в кабинете настроек в боковой панели выбрать пункт «Языки».  
   ![](https://howto.aqua-delivery.com/wp-content/uploads/2020/04/Screenshot_7.png)

2. На странице будут показаны доступные для перевода языки.  
![](https://howto.aqua-delivery.com/wp-content/uploads/2020/04/Screenshot_8.png)

На данный момент по умолчанию стоит русский язык.  
![](https://howto.aqua-delivery.com/wp-content/uploads/2020/04/Screenshot_9.png)

3. Для создания нового языка, необходимо нажать кнопку «Создать» в левом верхнем углу.

![](https://howto.aqua-delivery.com/wp-content/uploads/2020/04/Screenshot_10.png)

4. Перед вами откроется страница для заполнения.

![](https://howto.aqua-delivery.com/wp-content/uploads/2020/04/Screenshot_11.png)

Прописываем необходимые данные:

* Название —  например, «Чешский».
* Выбираем код — короткие алфавитные коды, разработанные для представления языков в обработке данных и коммуникациях.
* Последовательность — в этой последовательности можно будет их показывать при выборе.
* Активность — включает возможность выбора языка для пользователей. Пока перевод на новый язык в процессе перевода — убираем галочку.
* В любой неопределённой ситуации будет выбираться язык «По умолчанию» как наиболее вероятный для пользователей приложения.

5. В самом низу нажимаем «Сохранить».

### Перевод на другой язык

На данный момент для перевода на другой язык доступен следующий функционал:  
— товары (название, описание, объем);  
— наборы товаров (название, описание);  
— настройки (политика конфиденциальности, надписи, описание системы лояльности для приложения и т. д.)  
— офисы продаж (название, описание, адрес, график работы);

Рассмотрим пример перевода товара на другой язык.

1. Для начала заходим в левом боковом меню в раздел «Товары», выбираем товар и нажимаем на редактирование (карандашик справа).
2. В редактировании товара сверху во вкладке с языками появился английский язык, у которого мы заранее включили активность.

![](https://howto.aqua-delivery.com/wp-content/uploads/2020/04/Screenshot_12.png)

3. Нажимаем на вкладку с английским языком и вручную вводим данные на этом языке.  
![](https://howto.aqua-delivery.com/wp-content/uploads/2020/04/Screenshot_13.png)

4. Сохраняем данные.

5. Заходим в настройки телефона и включаем английский интерфейс. Затем открываем мобильное приложение. Интерфейс отображается на английском языке и товары тоже.  
![](https://howto.aqua-delivery.com/wp-content/uploads/2020/04/photo_2025-03-19_21-58-59.jpg)

Для корректного отображения всего контента необходимо перевести весь функционал в кабинете настроек.
