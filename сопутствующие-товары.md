# Сопутствующие товары — База знаний Aqua Delivery

Для добавления сопутствующего товара необходимо выполнить следующие действия:

1. Зайдите в **Кабинет настроек** *(admin.доменвашейкомпании.aqua-delivery.ru)*.
2. Выберите тот товар, к которому нужно добавить сопутствующие товары. Нажмите кнопку «Редактировать».![](https://aqua-delivery.ru/howto/wp-content/uploads/2020/03/pasted-image-0.png)
3. В форме редактирования товара находим поле «Сопутствующие товары». Нажмите «Выбрать товары».  
   ![](https://aqua-delivery.ru/howto/wp-content/uploads/2020/03/pasted-image-0-1.png)
4. В появившемся окне выберите товары, которые хотите прикрепить. Количество сопутствующего товара ограничено лишь вашим ассортиментом. ![](https://aqua-delivery.ru/howto/wp-content/uploads/2020/03/pasted-image-0-2.png)Например:![](https://aqua-delivery.ru/howto/wp-content/uploads/2020/03/pasted-image-0-3.png)
5. Сохраните.![](https://howto.aqua-delivery.com/wp-content/uploads/2020/03/photo_2026-01-22_09-11-14.jpg)
6. После добавления товара можно проверить его отображение в приложении.  
   ![](https://howto.aqua-delivery.com/wp-content/uploads/2020/03/image10.png)

При добавлении нескольких товаров в корзину, у которых есть сопутствующие товары, в поле «С этим товаром обычно берут» будут отображаться все сопутствующие товары выбранных товаров одной горизонтальной лентой.
