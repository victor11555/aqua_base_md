# Импорт индивидуальных цен через Excel — База знаний Aqua Delivery

**Шаги:**

1. Скачайте экземпляр Excel-файла или сопоставьте ваш Excel файл так, чтобы порядок столбцов совпадал с примером из экземпляра.
2. Передайте файл для импорта на проверку специалистам технической поддержки AquaDelivery. В случае возникновения вопросов специалисты отправят вам документ на доработку.
