# Возможности бесплатного сайта от Aqua Delivery — База знаний Aqua Delivery

Каждый партнер Aqua Delivery получает бонусом сайт, который облегчает продвижение приложения для заказа воды. У сайта есть много особенностей и инструментов для рекламы.

1. На сайте есть виджет для заказа, который полностью интегрирован с мобильным приложением. При желании, вы можете установить такой же на своем сайте.
2. По всему сайту есть информация о мобильном приложении, включая ссылки на его скачивание.
3. На базе сайта работает треккинговая система для отслеживания рекламной статистики.
4. У сайта есть функция редиректа. Другими словами (что такое редирект), переходя по ссылке с редиректом, клиент будет попадать на скачивание вашего приложения в нужный магазин приложений в зависимости от его платформы. С iOS в App Store, с Android в Google Play.
