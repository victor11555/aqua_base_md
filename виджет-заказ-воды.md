# Виджет «Заказ воды» — База знаний Aqua Delivery

Заказать воду можно не только с приложения, но и с сайта.

Для этого откройте ваш сайт и найдите кнопку в виде корзины:  
![](https://aqua-delivery.ru/howto/wp-content/uploads/2020/03/2.png)

Откроется виджет для заказа воды:  
![](https://aqua-delivery.ru/howto/wp-content/uploads/2020/03/3.png)

Выбрав позиции, которые нужны, нажимаем кнопку «Оформить»:  
![](https://aqua-delivery.ru/howto/wp-content/uploads/2020/03/4.png)

Заполняем данные для оформления заказа: ФИО, номер телефона, адрес, способ оплаты и дату доставки:  
![](https://aqua-delivery.ru/howto/wp-content/uploads/2020/03/5.png)

![](https://aqua-delivery.ru/howto/wp-content/uploads/2020/03/6.png)

После нажатия на кнопку «Заказать», виджет обновится, а ваш заказ поступит к оператору:  
![](https://aqua-delivery.ru/howto/wp-content/uploads/2020/03/7.png)
