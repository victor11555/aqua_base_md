# Что нужно делать, если клиент оплатил заказ, но в нем нужно что-то поменять — База знаний Aqua Delivery

- **Выполнить возврат онлайн-заказа и создать новый.**

1. Перейдите в кабинет заказов в раздел **Онлайн оплата.![](https://howto.aqua-delivery.com/wp-content/uploads/2023/11/photo_2026-01-26_11-30-50.jpg)**
2. Кнопка «Возврат оплаты» доступна пользователям с правами администратор. Для пользователя с ролью оператор данный функционал недоступен. Найдите заказ, у которого нужно провести возврат и нажмите на кнопку возврата. *!!!Важный момент, если в поле* *СБП* *стоит галочка, то возврат средств на данном шаге окончен.![](https://howto.aqua-delivery.com/wp-content/uploads/2023/11/photo_2026-01-26_11-33-49.jpg)*
3. Если у вас подключена интеграция с OFD.ру, то после совершения возврата автоматически формируется чек возврата, который можно будет посмотреть в заказе, в разделе чеки. *!!!Важный момент, если оплата была через Эквайринг, то возврат средств нужно произвести через личный кабинет банка.![](https://howto.aqua-delivery.com/wp-content/uploads/2023/11/3333.png)*

Если у вас ЮКасса, то перейдите в личный кабинет по ссылке.

1. Найдите нужный заказ для возврата и нажмите кнопку возврата (**Ищите по сумме заказа и времени отправки и если есть по почте**). Если у вас Сбербанк, то перейдите в личный кабинет по ссылке.![](https://howto.aqua-delivery.com/wp-content/uploads/2023/11/4444-1.png)
2. Перейдите в раздел «Транзакции», найдите нужный заказ для возврата и перейдите в него (Ищите по сумме заказа и времени отправки и если есть по почте).![](https://howto.aqua-delivery.com/wp-content/uploads/2023/11/5555.jpg)
3. Нажмите кнопку «Возврат» в правом верхнем углу.![](https://howto.aqua-delivery.com/wp-content/uploads/2023/11/6666.jpg)

Далее Вы можете создать новый заказ или попросить об этом Вашего клиента.

- **Если заказ был частично доставлен.**

*Вернуть часть суммы за заказ невозможно. Можем предложить вам три способа решения ситуации:*

*— договориться с клиентом и довезти недостающую часть заказа;*

*— выполнить полный возврат заказа, после попросить клиента сделать новый заказ (с товарами, которые ваш курьер уже доставил) (пункт 1 инструкции);*

*— начислить клиенту сумму недоставленной части заказа баллами лояльности.*

Вы можете начислять баллы клиенту самостоятельно, для этого:  
1. Войдите в Кабинет заказов.

2. Выберите раздел “Кешбэк”.

![](https://howto.aqua-delivery.com/wp-content/uploads/2023/11/photo_2026-01-26_11-37-54-1.jpg)

3. На открывшейся странице нажмите “Новая запись”.![](https://howto.aqua-delivery.com/wp-content/uploads/2023/11/photo_2026-01-26_11-46-51.jpg)
4. Заполните поля и нажмите “Сохранить”.       ![](https://howto.aqua-delivery.com/wp-content/uploads/2023/11/2026-01-26_11-48-40.png)

- «Пользователь» — введите имя пользователя, которому хотите начислить баллы.
- «Заказ» (необязательно) — если хотите связать начисление баллов с каким-либо заказом. Влияет исключительно на отображение в истории.
- «Причина начисления» — выберите причину начисления (влияет на отображение в истории). «Заказ» — начисление за заказ. «Приглашение в приложение» — заказ за приглашение друга. «Депозит» — внесение баллов по какой-либо иной причине.
- «Комментарий» (необязательно) — комментарий к внесению баллов, не отображается для клиента.
- «Значение» — количество баллов, которое вы хотите добавить клиенту.  
  Вы также можете списать баллы у клиента — для этого в поле «Значение» введите отрицательную величину, например -100.

Обращаем внимание, что обязательные поля для заполнения обозначены красной звездочкой.

В случае, если ни один из вариантов не подошел для решения ситуации — обращайтесь в техническую поддержку Aqua Delivery!
