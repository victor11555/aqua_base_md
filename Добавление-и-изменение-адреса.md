# Добавление и изменение адреса — База знаний Aqua Delivery

Чтобы добавить адрес, нажмите кнопку рядом с текстом «Адрес доставки», после чего вам откроется окно заполнения информации о новом адресе. Есть два способа добавить адрес:

1. Вручную написать полный адрес. После этого выберите нужный адрес из выпадающего списка. Нажмите кнопку «Сохранить».![](https://howto.aqua-delivery.com/wp-content/uploads/2023/08/12.gif)
2. Выбрать адрес на карте. Для этого в строку «Город, улица, дом» внесите любое значение, после чего в выпадающем списке выберите пункт «Выбор на карте». Нажмите кнопку «Сохранить».![](https://howto.aqua-delivery.com/wp-content/uploads/2023/08/13-1.gif)

Изменить существующий адрес можно двумя способами:

1. С помощью карты. Для этого нажмите на кнопку в виде карандаша, после чего нажмите кнопку почти в самом низу окна редактирования в виде гео-метки. Выберите нужный адрес и нажмите кнопку «Сохранить».  
   ![](https://howto.aqua-delivery.com/wp-content/uploads/2023/08/14.gif)
2. С помощью строки «Город, улица, дом». Для этого нажмите на кнопку в виде карандаша, после чего введите нужное значение адреса в строку «Город, улица, дом» и выберите значение из выпадающего списка, после чего нажмите кнопку «Сохранить».

**Важно! Строки «Населенный пункт», «Улица», «Дом», «Зона доставки» и «Район доставки» нельзя изменить вручную. Чтобы изменить их значение, воспользуйтесь описанными выше способами.**
