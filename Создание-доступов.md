# Доступы — База знаний Aqua Delivery

**Доступ к кабинету заказов**

В **Кабинете заказов** оператор может просматривать, обрабатывать заказы.

![](https://howto.aqua-delivery.com/wp-content/uploads/2022/06/5.jpg)

Можно ограничить работу оператора в определенном городе, то есть он сможет создавать и принимать заказы только в выбранном городе.

Доступ создается на стороне Aqua Delivery, для его создания, напишите в техподдержку.

**Доступ к кабинету маркетолога**

Главная страница кабинета маркетолога:

![](https://howto.aqua-delivery.com/wp-content/uploads/2022/06/Screenshot_2-2.png)

В **Кабинете маркетолога** собраны следующие отчеты:

1. Количество выполненных заказов
2. Количество заработанных денег
3. Активные клиенты
4. Установки и активность клиентов
5. Купленные товары
6. Заказы по часам
7. Потерянные/вернувшиеся клиенты
8. Учёт баллов системы лояльности
9. Оценка приложения

Также представлены следующие инструменты:

1. Индивидуальный пуш
2. Трекинговая система
3. Пуш-расылки

**Доступ к кабинету настроек**

Для кабинета настроек создается доступ администратора, он также будет иметь доступ к остальным кабинетам.
