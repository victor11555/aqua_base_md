# Контур работы склада — База знаний Aqua Delivery

Складской учет состоит из нескольких контуров:

- Пополнение склада;

- Списание товаров со склада после продажи;

- Прочие списания товаров.

Чтобы учет склада работал стабильно, необходимо придерживаться подхода, когда любое списание товара и поступление, фиксируется.

**Пополнение склада**

Приход товара создается следующими документами:

- Приходная накладная;

- Оприходование запасов;

- Документ производства;

- Ввод начальных остатков.

В основном работа предстоит с приходными накладными, документ, который создается на основании товарной накладной, которую вам передал ваш поставщик, и документами производства.

**Списание товаров со склада после продажи**

Списание товара со склада после продажи будет производиться автоматически. В качестве основания для списания товара служат 2 документа:

- Отчет о розничных продажах для физических лиц;

- Расходная накладная для юридических лиц;

Обратите внимание, чтобы произошло автоматическое списание товара со склада, на все заказы обязательно должен быть назначен водитель. За каждым водителем закрепляется касса и склад, с которого будет списан товар при завершении заказа (В вашем случае таким складом будет являться склад реализации, что означает товары будут списываться только с него. За водителем строго закрепляется склад, с которого будет производиться списание. В случае, если водитель будет доставлять заказы, которые были погружены на другом складе, то списание все равно произойдет со склада реализации).

**Отчеты**

Отслеживать работу склада вы можете с помощью отчетов, которые находятся в разделе «Склад» → «Отчеты» → «Склад».  
![](https://howto.aqua-delivery.com/wp-content/uploads/2025/06/image-5-2.png)

**Как делать списание товаров со склада**

Благодаря списанию товаров со склада можно поддерживать корректный учет остатков на складах, отслеживать финансовые потери и др.

Для списания товаров со склада нужно:

1. Переходим в «Склад» → «Списания».  
![](https://howto.aqua-delivery.com/wp-content/uploads/2025/06/image-6-2.png)

2. В данном разделе можно ознакомиться со списаниями запасов, а также создать новое списание. Создаем списание товаров со склада.

![](https://howto.aqua-delivery.com/wp-content/uploads/2025/06/image-7-2.png)

3. В открывшейся форме выбираем корреспонденцию (причину списания) и склад, с которого будет произведено данное списание.  
![](https://howto.aqua-delivery.com/wp-content/uploads/2025/06/image-8-2.png)

4. Далее добавляем товар, который нужно списать, и его количество. Также в комментариях к списанию запасов можно указать причину. Проводим и закрываем.  
![](https://howto.aqua-delivery.com/wp-content/uploads/2025/06/image-9-2.png)

5. Если среди списанной продукции есть маркируемая продукция, создаем документ вывода из оборота.  
![](https://howto.aqua-delivery.com/wp-content/uploads/2025/06/image-10-2.png)

6. Переходим в документ вывода из оборота. Заполняем наименование документа, указав «Списание» и в разделе товары, через подбор и проверку маркировки добавляем маркировку, которая выведется из оборота (Маркировку вы можете получить с помощью подключенного ТСД, с помощью приложения «Честный знак бизнес» или с помощью любого другого приложения для сканирования кодов).  
![](https://howto.aqua-delivery.com/wp-content/uploads/2025/06/image-11-2.png)![](https://howto.aqua-delivery.com/wp-content/uploads/2025/06/image-12-2.png)

7. Передаем документ в ЧЗ.  
![](https://howto.aqua-delivery.com/wp-content/uploads/2025/06/image-13-2.png)

После проведения списания со склада, в списке списаний отобразится новое списание и товар, который списывался, будет вычтен из остатков. То есть в отчете об остатках на складах, остатки данного товара будут отображаться с учетом проведенного списания.
