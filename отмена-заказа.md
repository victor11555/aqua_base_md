# Отмена заказа из приложения — База знаний Aqua Delivery

Для того, чтобы отменить заказ из приложения, необходимо:

1. В нижнем меню выбрать пункт **«****Заказы».**  
   ![](https://howto.aqua-delivery.com/wp-content/uploads/2020/08/image36.png)
2. Выберите заказ, который хотите отменить.
3. Внизу располагается кнопка **«Отменить»**, на нее необходимо нажать.  
   ![](https://howto.aqua-delivery.com/wp-content/uploads/2020/08/photo_2025-03-13_21-59-10.jpg)
4. Выберите причину отмены заказа в выпадающем списке или напишите ее самостоятельно.  
   ![](https://howto.aqua-delivery.com/wp-content/uploads/2020/08/photo_2025-03-13_22-04-40.jpg)  
   ![](https://howto.aqua-delivery.com/wp-content/uploads/2020/08/photo_2025-03-13_22-06-06.jpg)
5. Затем завершите заказ нажатием на кнопку **«****Подтвердить».  
   ![](https://howto.aqua-delivery.com/wp-content/uploads/2020/08/photo_2025-03-13_22-11-33.jpg)**
