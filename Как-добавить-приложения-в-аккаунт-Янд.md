# Как добавить приложения в аккаунт Яндекс.Директ — База знаний Aqua Delivery

1. Переходим в раздел Библиотека → Мобильные приложения.

![](https://howto.aqua-delivery.com/wp-content/uploads/2022/07/image4.png)

2. Нажимаем Добавить приложение.

![](https://howto.aqua-delivery.com/wp-content/uploads/2022/07/image3.png)

3. Вставляем ссылку на приложение в сторе. Ссылки можно взять в кабинете партнера.

![](https://howto.aqua-delivery.com/wp-content/uploads/2022/07/image1.png)

4. В следующее поле вставляем трекинговую ссылку соответствующую платформе приложения.

![](https://howto.aqua-delivery.com/wp-content/uploads/2022/07/image2.png)

5. Сохраняем и добавляем по тому же принципу второе приложение.
