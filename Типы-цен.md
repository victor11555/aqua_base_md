# Типы цен — База знаний Aqua Delivery

Для того, чтобы добавить типы цен, выполните следующие действия:

1. Войдите в **Кабинет настроек** (admin.доменное\_имя\_вашей\_компании.aqua-delivery.ru).

2. В боковом меню нажмите на пункт **«****Справочники»** и выберите раздел **«****Типы цен».**

![](https://howto.aqua-delivery.com/wp-content/uploads/2022/05/Screenshot_11.png)

3. Откроется страница «**Типов цен»**.

![](https://howto.aqua-delivery.com/wp-content/uploads/2022/05/Screenshot_1-2.png)

4. На этой странице вы можете редактировать приоритет цены товара. Эта функция нужна для того, чтобы тип цены корректно отображался в приложении. Соответственно, в заказ будет подставляться та цена, приоритет у которой выше, чем у других.

Для изменения приоритета вам нужно выбрать интересующую вас строку и напротив нее, в столбце «Приоритет», изменить цифру на подходящую.

Пример: ![](https://howto.aqua-delivery.com/wp-content/uploads/2022/05/2025-11-17_11-26-27.png)

5. На этой странице вы включаете только активность определенного функционала, его настройка осуществляется на другой странице. Чтобы включить тип цены, поставьте галочку в столбце **«Активность»**.

![](https://howto.aqua-delivery.com/wp-content/uploads/2022/05/Screenshot_2-2.png)

6. Настройка типов цен:

1.Индивидуальная цена.

Настраивается по инструкции

2. Цена при самовывозе (клиент в мобильном приложении выбрал **Самовывоз**).

3.Цена выходного дня (клиент оформил заказ в выходной день).

4. Цена по карте и нал (клиент выбрал один из двух способом оплаты).

5. Цена для физ лица (клиент имеет тип лица — физ. лицо)

6. Цена в зависимости от источника.

7. Чтобы настроить типы цен перейдите в раздел **«****Товары».**

![](https://howto.aqua-delivery.com/wp-content/uploads/2022/05/Screenshot_5-1.png)

8. Затем выберите нужный товар и перейдите в раздел **«Типы цен».** Для этого нажмите на значок доллара напротив нужного товара.

![](https://howto.aqua-delivery.com/wp-content/uploads/2022/05/Screenshot_6.png)

9. На этой странице вы можете указать стоимость товара в зависимости от условия.

![](https://howto.aqua-delivery.com/wp-content/uploads/2022/05/Screenshot_7.png)
