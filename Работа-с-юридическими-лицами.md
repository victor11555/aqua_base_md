# Работа с юридическими лицами — База знаний Aqua Delivery

Информация обо всех юридических лицах содержится в разделе «Организации».

1. Для того, чтобы перейти в этот раздел вам необходимо в кабинете заказов в боковом меню выбрать «Организации».

![](https://howto.aqua-delivery.com/wp-content/uploads/2024/07/о1.jpg)

2. Здесь содержатся следующие данные обо всех юридических лицах:

- Название организации;
- ИНН;
- КПП.

![](https://howto.aqua-delivery.com/wp-content/uploads/2024/07/о3.jpg)

**Прикрепление клиентов к организации.**

1. К любой организации можно прикрепить несколько ваших клиентов. Для этого перейдите в карточку клиента и нажмите на значок плюса возле поля «Организация».

![](https://howto.aqua-delivery.com/wp-content/uploads/2024/07/о4.jpg)

2. На открывшейся странице укажите название и адрес организации (на данный адрес будут осуществляться заказы), к которой необходимо прикрепить клиента. Нажмите «Сохранить».

![](https://howto.aqua-delivery.com/wp-content/uploads/2024/07/о5.jpg)

**Создание заказа для организации**

- **Если у одной организации несколько клиентов.**

1. Для создания заказа в разделе «заказы» выберите «Новый заказ».

![](https://howto.aqua-delivery.com/wp-content/uploads/2024/07/о6.jpg)

2. В открывшемся окне установите фильтр «организация» и введите название организации.

![](https://howto.aqua-delivery.com/wp-content/uploads/2024/07/о7.jpg)

3. Если к организации привязано несколько клиентов, то в выпадающем списке у вас появится возможность выбрать клиента, которому необходимо доставить заказ (если за организацией числится один клиент, то он будет выбран автоматически).

![](https://howto.aqua-delivery.com/wp-content/uploads/2024/07/о8.jpg)

4. После выбора клиента вы сможете выбрать адрес из уже существующих или добавить новый, нажав на «Добавить новый адрес» в выпадающем списке:

![](https://howto.aqua-delivery.com/wp-content/uploads/2024/07/о9.jpg)

- **Если один клиент прикреплен к нескольким организациям (вы можете это посмотреть в карточке клиента).**

![](https://howto.aqua-delivery.com/wp-content/uploads/2024/07/о10.png)

1. Если у вас есть клиент, закрепленный сразу за несколькими организациями, при создании заказа можно выбрать одну из них. Для этого в разделе «заказы» выберите «Новый заказ».

![](https://howto.aqua-delivery.com/wp-content/uploads/2024/07/о11.jpg)

2. В открывшемся окне установите фильтр «везде» и введите данные клиента. Для выбора конкретной организации нажмите на «адрес» и выберите адрес, который закреплен за организацией, название организации также зафиксировано рядом с адресом.

![](https://howto.aqua-delivery.com/wp-content/uploads/2024/07/11111-2.jpg)
