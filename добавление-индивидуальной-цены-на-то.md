Вы можете настроить индивидуальные цены на товар для каждого клиента.

1. Войдите в **Кабинет настроек** и слева в боковом меню выберите раздел **«****Клиенты»**.

![](https://howto.aqua-delivery.com/wp-content/uploads/2020/08/Screenshot_3.png)

2. Перед вами откроется страница со всеми вашими клиентами. Для того чтобы установить индивидуальную цену клиенту, необходимо перейти в его карточку через редактирование (карандашик справа).

![](https://howto.aqua-delivery.com/wp-content/uploads/2020/08/Screenshot_4.png)

3. Откроется страница **«****Изменение клиента»**. В нижней части страницы есть раздел **«****Индивидуальная цена на товары**.» Нажмите на кнопку с *плюсом*, чтобы создать новую запись об индивидуальной цене. В появившейся форме выберите товар, на который будет распространяться индивидуальная цена, саму цену и поставьте галочку в поле **Активность**, чтобы активировать индивидуальную цену.  
![](https://howto.aqua-delivery.com/wp-content/uploads/2020/08/Screenshot_5.png)

4. Внизу нажмите **«****Сохранить»**.

Теперь для клиента будет отображаться именно та цена, которую вы указали в его карточке.

Чтобы удалить индивидуальную цену, нажмите на красную кнопку с крест*иком* правее цены и нажмите **«****Сохранить»**.

![](https://howto.aqua-delivery.com/wp-content/uploads/2020/08/Screenshot_6.png)
