# Карточка клиента — База знаний Aqua Delivery

Из этой инструкции вы узнаете:

- **Как перейти в карточку клиента;**
- **Что есть в карточке клиента;**
- **Как изменить информацию о клиенте;**
- **Как добавить и изменить адрес.**

**Как перейти в карточку клиента?**

1. В разделе «Заказы» нажмите на ФИО клиента, после чего откроется его карточка.
2. Перейдите в раздел «Клиенты» и нажмите на ФИО клиента, после чего откроется карточка.

**Что есть в карточке клиента?**

1. Информация о клиенте: ID, ФИО, Email, телефон.![](https://howto.aqua-delivery.com/wp-content/uploads/2023/08/photo_2026-01-20_13-53-53-1.jpg)
2. Информация об адресах клиента.  
   ![](https://howto.aqua-delivery.com/wp-content/uploads/2023/08/2023-08-18_16-45-31.png)
3. Информация об активности клиента: Общая сумма, количество баллов, количество сделанных заказов. **Важный момент! Сумма и Заказы не учитывают отмененные заказы.  
   ![](https://howto.aqua-delivery.com/wp-content/uploads/2023/08/2023-08-18_17-00-27.png)**
4. Вся история операций, связанных с клиентом.  
   В разделе «Вся история» находятся все операции.  
   В разделе «Заказы» можно увидеть все сделанные клиентом заказы. Если нажать на заказ, то вы сможете перейти в карточку клиента.  
   В разделе «Комментарии» можно посмотреть историю комментариев, оставляемых клиентом.  
   В разделе «Оператор» описаны все действия, связанные с заказами, которые совершал оператор.  
   ![](https://howto.aqua-delivery.com/wp-content/uploads/2023/08/Диаграмма-возможностей.png)
5. Чат с клиентом. Чтобы перейти в чат с клиентом, нажмите на кнопку в виде чат-облака.  
   ![](https://howto.aqua-delivery.com/wp-content/uploads/2023/08/2023-08-18_17-21-20.png)
6. Периодичность заказов.  Периодичность определяется по 5 последним заказам.  
   ![](https://howto.aqua-delivery.com/wp-content/uploads/2023/08/2023-08-18_17-28-49.png)
7. Текущий долг по таре. Можно скачать отчет по истории тары в формате XLS.  
   ![](https://howto.aqua-delivery.com/wp-content/uploads/2023/08/2023-08-18_17-31-24.png)

**Изменение информации о клиенте**

Чтобы изменить информацию о клиенте, нажмите на кнопку редактирования в виде карандаша, после чего вам откроется окно редактирования. Измените нужную вам информацию, затем нажмите кнопку «Сохранить». **Важно! Номер телефона и адрес электронной почты нельзя заменить на те, которые уже используются.**

![](https://howto.aqua-delivery.com/wp-content/uploads/2023/08/11.gif)
