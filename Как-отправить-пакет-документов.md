# Как отправить пакет документов — База знаний Aqua Delivery

*В 1С Aqua Delivery работа по ЭДО проходит с помощью оператора ЭДО Калуга Астрал. Также важно учитывать, что при работе с ЭДО в компьютер обязательно должна быть вставлена электронная цифровая подпись (ЭЦП). Без данной подписи документы не будут подписаны и отправлены контрагенту.*

Пакетная отправка документов — инструмент, позволяющий быстро и удобно направить КА несколько документов (пакет) с помощью ЭДО. Также важно понимать, что отправка пакета документов выгоднее, чем отправка документов поэкземплярно.

Для пакетной отправки документов нужно:  
  
1. Переходим в раздел «Текущие дела ЭДО».  
![](https://howto.aqua-delivery.com/wp-content/uploads/2025/07/Скриншот-29-07-2025-154543.jpg)

2. Выбираем нужный документ (например, расходная накладная) и просматриваем его.  
![](https://howto.aqua-delivery.com/wp-content/uploads/2025/07/Скриншот-29-07-2025-154702.jpg)

3. С помощью «Добавить в пакет» добавляем необходимые файлы. Хранилищем таких документов могут быть:  
— файл с компьютера;  
— файл с хранилища;  
— документ информационной базы — то есть любой документ из 1С.  
![](https://howto.aqua-delivery.com/wp-content/uploads/2025/07/Скриншот-29-07-2025-154801.jpg)  
Добавленные документы отобразятся в строке «Документ учета». Если в пакет добавляется более 3 документов, их можно будет просмотреть с помощью кнопки «Еще».   
![](https://howto.aqua-delivery.com/wp-content/uploads/2025/07/Скриншот-29-07-2025-154938.jpg)  
  
4. Подписываем и отправляем пакет документов.  
![](https://howto.aqua-delivery.com/wp-content/uploads/2025/07/Скриншот-29-07-2025-155126.jpg)  
  
5. Для просмотра пакета документов нужно распаковать пакет. Для этого переходим в «Текущие дела ЭДО» в папку «Распаковать» и распаковываем пакет документов.  
![](https://howto.aqua-delivery.com/wp-content/uploads/2025/07/Скриншот-29-07-2025-155308.jpg)
