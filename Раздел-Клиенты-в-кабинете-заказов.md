# Раздел «Клиенты» в кабинете заказов — База знаний Aqua Delivery

Чтобы перейти в раздел Клиенты нужно в кабинете заказов, в боковом меню выбрать Клиенты.

![](https://howto.aqua-delivery.com/wp-content/uploads/2022/06/19.jpg)

В разделе содержится информация о клиентах.

![](https://howto.aqua-delivery.com/wp-content/uploads/2022/06/20.jpg)

По каждому клиенту представлены следующие данные:![](https://howto.aqua-delivery.com/wp-content/uploads/2022/06/21.jpg)

1. ID.
2. ФИО (нажав на имя клиента можно перейти в карточку клиента).
3. Абонентский номер.
4. Номер телефона.
5. E-mail.
6. Количество доставленных заказов.
7. Количество баллов (на счету).
8. Адрес (адреса, внесенные в приложение клиента).
9. Дата последнего заказа.

Можно найти клиента через поисковые строки, по вышеперечисленным пунктам.

![](https://howto.aqua-delivery.com/wp-content/uploads/2022/06/22.jpg)

Также в этом разделе представлен следующий функционал:

- Возврат клиентов, которые перестали делать заказы. Подробнее в этой инструкции![](https://howto.aqua-delivery.com/wp-content/uploads/2022/06/photo_2026-01-23_12-00-27.jpg)
- Выгрузка клиентов с индивидуальными ценами.![](https://howto.aqua-delivery.com/wp-content/uploads/2022/06/photo_2026-01-23_13-17-48.jpg)

После нажатия на кнопку на компьютер автоматически загрузится файл Excel со списком клиентов и их индивидуальной ценой.

![](https://howto.aqua-delivery.com/wp-content/uploads/2022/06/Screenshot_13.png)

- Выгрузить цены по всем товарам.

После нажатия на кнопку на компьютер автоматически загрузится файл Excel со списком товаров, ценами на них и от чего зависит данная цена.

![](https://howto.aqua-delivery.com/wp-content/uploads/2022/06/Screenshot_14.png)

- Адреса без координат.

С помощью этой функции можно установить координаты для адреса. Подробнее в инструкции
