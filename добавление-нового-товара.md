# Добавление нового товара — База знаний Aqua Delivery

**Для добавления нового товара:**

1. Зайдите в **Кабинет настроек** *(admin.домен\_вашей\_компании.aqua-delivery.ru).*
2. Выберите в боковом меню пункт **«****Товары»**.

**![](https://howto.aqua-delivery.com/wp-content/uploads/2020/06/Add1.png)**

3. Выберите товары из нужной категории.  Для этого нажмите на **«****Товары»** в столбце **«****Товары в категории»**.![](https://howto.aqua-delivery.com/wp-content/uploads/2020/06/Add2.png)

4. На странице «**Товары**» нажмите кнопку «**Создать**».  
![](https://howto.aqua-delivery.com/wp-content/uploads/2020/06/Add3.png)

5. Откроется страница создания товара. Последовательно заполняйте все поля:

![](https://howto.aqua-delivery.com/wp-content/uploads/2020/06/Add4.png)

- **Название** — название товара, которое отображается в приложении.

![](https://howto.aqua-delivery.com/wp-content/uploads/2020/06/Add5.png)

- **Описание** — описание товара, которое отображается при нажатии на товар в приложении.

![](https://howto.aqua-delivery.com/wp-content/uploads/2020/06/Add6.png)

- **Объем** — объем воды или бутыли, отображается при нажатии на товар в приложении. В этом поле можно также указать вес товара, включая единицы измерения (например, литры, граммы и килограммы).

![](https://howto.aqua-delivery.com/wp-content/uploads/2020/06/Add7.png)

- Обязательно включите пункт **«****Отображается»**, чтобы ваш товар показывался в приложении.

![](https://howto.aqua-delivery.com/wp-content/uploads/2020/06/Add8.png)

- **Доступен при экспрессе**— товар будет доступен при выборе экспресс-доставки, если настроен интервал.

![](https://howto.aqua-delivery.com/wp-content/uploads/2020/06/Add9.png)

- **Включить маркировку** — при оформлении заказа после активации данной функции, в курьерском приложении будет доступен функционал для сканирования маркировочных кодов.

![](https://howto.aqua-delivery.com/wp-content/uploads/2020/06/Add10.png)

- **GTIN** — уникальный код товара, который находится внутри маркировочного кода и позволяет идентифицировать товар.

![](https://howto.aqua-delivery.com/wp-content/uploads/2020/06/Add11.png)

- **Базовый тип** влияет на логику принадлежности определённой позиции.

![](https://howto.aqua-delivery.com/wp-content/uploads/2020/06/Add12.png)

В системе предусмотрены следующие базовые типы:

- - **Продукт** — в случае, если некоторое наименование является товаром.
  - **Услуга** — для всего остального, что относится к форме деятельности (например, чистка кулера).

- **Категория** — в какой категории (вкладке в приложении) будет отображаться товар.

![](https://howto.aqua-delivery.com/wp-content/uploads/2020/06/Add13.png)

- **Тип товара** влияет на логику работы товара.

![](https://howto.aqua-delivery.com/wp-content/uploads/2020/06/Add14.png)

В системе предусмотрены следующие типы товаров:

1. - **Вода 19 л** — товары, работающие с возвратной тарой.
   - **Оборудование** —  дополнительные товары, к которым относятся помпы, кулеры и т.п.
   - **Напитки** — вода объёмом менее 19 л.

- **Минимальное количество в заказе** — минимальное количество товара, которое можно заказать. Например, если стоит значение 3, то в корзину добавится сразу три единицы товара без возможности уменьшения количества ниже, чем указанное значение.

![](https://howto.aqua-delivery.com/wp-content/uploads/2020/06/Add15.png)

- **Максимальное количество в заказе** — максимальное количество товара, которое можно заказать. Например, если стоит значение 10, то в корзину можно добавить до десяти единиц товара без возможности увеличения количества выше, чем указанное значение.

![](https://howto.aqua-delivery.com/wp-content/uploads/2020/06/Add16.png)

- **Сопутствующие товары** — какие товары будут предлагаться к покупке вместе с этим товаром (подробнее читайте здесь).

![](https://howto.aqua-delivery.com/wp-content/uploads/2020/06/Add17.png)

- **Отображать в зонах доставки** — в каких зонах доставки (населённых пунктах) будет отображаться данный товар.

![](https://howto.aqua-delivery.com/wp-content/uploads/2020/06/Add18.png)

- **Цена** — стандартная цена за товар, применяющаяся, если нет других условий (цена за определенное количество товара, цена в определенном городе, индивидуальная цена). Подробнее об изменении цен в зависимости от условий читайте здесь.

![](https://howto.aqua-delivery.com/wp-content/uploads/2020/06/Add19.png)

- **Вознаграждение** — количество баллов в процентах от стоимости товара, которое получит клиент при покупке данного товара при включенной системе лояльности. Если поле будет пустым, то клиент не получит баллы за покупку этого товара.

![](https://howto.aqua-delivery.com/wp-content/uploads/2020/06/Add20-1.png)

- **Базовый товар** — поле для товара, который содержит несколько единиц другого товара (например, упаковка воды 0.5 л содержит 12 бутылок воды 0.5 л). Подробнее о данном функционале можете прочитать здесь.

![](https://howto.aqua-delivery.com/wp-content/uploads/2020/06/Add21.png)

- **Главное изображение** — изображение, которое отображается в ленте и при нажатии на товар, если отсутствуют дополнительные изображения. Формат главного изображения — один к одному (например, пятьсот на пятьсот), ограничение на размер — 1 мегабайт.

![](https://howto.aqua-delivery.com/wp-content/uploads/2020/06/Add22.png)

- **Дополнительное изображение** — изображения, которые отображаются при нажатии на товар. Изображения отображаются в случайном порядке. При наличии дополнительных изображений главное изображение не отображается при нажатии на товар. Формат изображений — один к одному (например, пятьсот на пятьсот), ограничение на размер — 1 мегабайт.

![](https://howto.aqua-delivery.com/wp-content/uploads/2020/06/Add23.png)

Не забудьте нажать **«****Сохранить»** после заполнения товара!

**Для редактирования, изменения цены или удаления существующего товара:**

1. Откройте **Кабинет настроек**.
2. В боковом меню выберите пункт **«****Товары»**.
3. На открывшейся странице выберите нужную категорию.
4. Нажмите на иконку с карандашом напротив нужного товара.  
   Для изменения цены в зависимости от условий нажмите на иконку валюты; для удаления товара нажмите на иконку с корзиной.

![](https://howto.aqua-delivery.com/wp-content/uploads/2020/06/Add24.png)
