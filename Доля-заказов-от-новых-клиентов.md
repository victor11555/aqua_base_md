# Доля заказов от новых клиентов — База знаний Aqua Delivery

В данной инструкции мы опишем, как посмотреть процент создания заказов от новых клиентов из общего числа созданных заказов.

В первую очередь вам потребуется перейти в кабинет оператора (Кабинет заказов). Далее выберите интересующий вас диапазон создания заказа.![](https://howto.aqua-delivery.com/wp-content/uploads/2024/01/Screenshot_1.jpg)

**Выбор диапазона создания заказа**

![](https://howto.aqua-delivery.com/wp-content/uploads/2024/01/2.jpg)

Теперь экспортируйте заказы, для этого нажмите кнопку экспорта в левом верхнем углу.

![](https://howto.aqua-delivery.com/wp-content/uploads/2024/01/3.jpg)

**Экспорт заказов**

![](https://howto.aqua-delivery.com/wp-content/uploads/2024/01/4.jpg)

**Экспорт заказов (Новый дизайн)**

В полученном файле найдите столбец «Первый заказ» и полностью его выделите (Нажмите на букву над столбцом).**![](https://howto.aqua-delivery.com/wp-content/uploads/2024/01/2024-01-11_20-59-06.png)**

**Выделение столбца «Первый заказ»**

Далее перейдите в раздел «Вставка» в верхнем меню и выберите «Рекомендуемые диаграммы», после чего откроется окно с одной диаграммой, которую нужно выбрать.![](https://howto.aqua-delivery.com/wp-content/uploads/2024/01/2024-01-11_21-01-40.png)

**Выбор диаграммы**

Далее нужно добавить в раздел «Фильтры» пункт «Источник», для этого просто перетащите его в нужный раздел, после чего в появившейся строке «Источник» в правом верхнем углу выберите «Приложение».

![](https://howto.aqua-delivery.com/wp-content/uploads/2024/01/16.gif)

**Добавление фильтрации**

![](https://howto.aqua-delivery.com/wp-content/uploads/2024/01/2024-01-11_21-08-17.png)

**Настройка фильтрации**

И наконец для большей наглядности выберите нужный диапазон и добавьте круговую диаграмму, в которой нужно настроить показ доль в процентах, для этого поставьте галочку рядом с пунктом «Метки данных» и перейдите в «Дополнительные параметры», где выберите пункт «Доли».![](https://howto.aqua-delivery.com/wp-content/uploads/2024/01/17.gif)
