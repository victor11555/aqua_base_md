# Как установить на мобильное устройство приложение для курьеров и авторизоваться в нем. — База знаний Aqua Delivery

*Инструкция предназначена для должности курьера (водителя) компании.*

*Благодаря данной инструкции вы можете узнать, как установить приложение для курьера на мобильное устройство и авторизоваться в нем.*

**Для установки курьерского приложения можно воспользоваться двумя способами:**   
1. Перейдите по ссылке https://democourier.app11.ru/get и установите приложение на мобильное устройство.

2. Перейдите в стор на мобильном устройстве (Play Market, App Store), введите в поисковой строке «Курьер Aqua Delivery». В появившемся списке приложений выберите необходимое и скачайте его на мобильное устройство.  
![](https://howto.aqua-delivery.com/wp-content/uploads/2024/01/photo_2024-01-25_14-43-31.jpg)

*Обратите внимание:*

*Если на мобильном устройстве нет Play Market, App Store, в этом случае следует обратиться к администратору компании и запросить apk файл для установки мобильного приложения. После получения apk файла, скачайте его на устройство.*

**Для авторизации в курьерском приложении необходимо выполнять действия по следующему алгоритму:**

1. Установите мобильное приложение на устройство.
2. Запросите данные для доступа в курьерское приложение у руководства компании.
3. После получения данных для доступа, перейдите в приложение. Вам откроется следующая форма, в которой требуется заполнить логин, (номер телефона) и пароль, которые предоставил руководитель.
