# Обзорный workspace — База знаний Aqua Delivery

![](https://howto.aqua-delivery.com/wp-content/uploads/2024/01/анализ1.png)

Обзорный workspace — это набор виджетов с данными отчетов AppMetrica. Другими словами, workspace — это виджеты с краткой выдержкой данных. Виджеты можно изменять, перемещать, удалять и добавлять.

Обзорный workspace полезен тем, что можно быстро посмотреть все нужные отчеты без деталей в одном окне.

Чтобы настроить виджеты, нужно нажать на кнопку “Виджеты” справа сверху:  
![](https://howto.aqua-delivery.com/wp-content/uploads/2024/01/анализ2.png)
