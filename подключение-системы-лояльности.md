# Подключение системы лояльности — База знаний Aqua Delivery

# Подключение

Система лояльности в системе Aqua Delivery состоит из трех основных частей:

- Кэшбек за покупки в приложении
- Инвайты (Пригласи друга)
- Промокоды

Чтобы подключить систему лояльности, напишите нам в Телеграм t.me/AquaDelivery\_support\_bot или на почту support@aqua-delivery.ru, и наши специалисты активируют её для вас.

# Настройка системы лояльности

Настройка системы лояльности происходит в **Кабинете настроек** (администратора) *admin.доменвашейкомпании.aqua-delivery.ru.* Для перехода к настройкам системы лояльности в боковом меню **Кабинета настроек** выберите раздел **«****Система лояльности»** и пункт **Настройки**.  
![](https://howto.aqua-delivery.com/wp-content/uploads/2020/09/Screenshot_3.png)

После этого вы перейдете на страницу настроек системы лояльности.  
![](https://howto.aqua-delivery.com/wp-content/uploads/2020/09/Screenshot_4-1.png)

1. **Включить систему лояльности для новых адресов** — если настройка активна, все новые создаваемые адреса по умолчанию участвуют в системе лояльности. Если выключена — не участвуют.
2. Если система лояльности была отключена, а потом включена, то клиенты, которые были зарегистрированы в этот промежуток времени не будут участвовать в системе лояльности.  
   *Также можно включить или отключить систему лояльности для конкретного клиента. Для этого в боковом меню выберите раздел **Клиенты** и нажмите на иконку карандаша для перехода к редактированию данных клиента.  
   ![](https://howto.aqua-delivery.com/wp-content/uploads/2020/09/Screenshot_5.png)  
   Пункт **Участвует в системе лояльности** определяет, включена ли для этого клиента система лояльности в принципе, пункт **Адреса, участвующие в системе лояльности** определяет, включена ли система лояльности для конкретных адресов.  
   ![](https://howto.aqua-delivery.com/wp-content/uploads/2020/09/Screenshot_6.png)*
3. Описание системы лояльности — текст, описывающий систему лояльности и отображающийся при нажатии на иконку балла в приложении. Рекомендуем вам использовать наш стандартный вариант текста без использования маркировки: **«Как получить бонусы?** Рядом с ценой каждого товара указано количество баллов, которое вы получите при покупке этого товара. Бонусы начисляются после успешной доставки заказа (за отменённые заказы и за заказы, полностью или частично оплаченные баллами, бонусы не начисляются). **Как потратить бонусы?**Оплачивайте покупки баллами при оформлении заказа через приложение. «При использовании функции маркировки текст в разделе **Как потратить бонусы?** необходимо поменять на: «Бонусы можно потратить при оформлении заказа через приложение.Для оплаты бонусами на финальной стадии оформления заказа поставьте галочку в квадратном поле, рядом с которым отображается сумма ваших баллов. При оплате заказа баллами минимальная стоимость, оплаченная деньгами, должна составить 1 рубль. При этом можно выбрать любой тип оплаты.»

   ![](https://howto.aqua-delivery.com/wp-content/uploads/2020/09/photo_2022-03-25_15-33-12.jpg)
4. Баллы при заказе — позволяет клиентам использовать накопленные баллы для оплаты заказов.

Есть два варианта списания баллов:

1. Частичное списание — при котором можно списать до 99% бонусов во время оформления заказа, в зависимости от количества бонусов у клиента.
2. Полное списание — при котором списать можно не менее 99% бонусов.

Настройка **Включить использование баллов**, когда активна, позволяет клиентам использовать баллы, когда же она не активна, то клиенты не могут использовать баллы для оплаты заказа (чтобы включить выберите Включить использование баллов).

Настройка **Возможность частичной оплаты** — если настройка выключена, то клиенты могут оплачивать заказ только целиком (например, заказ на триста рублей клиент сможет оплатить либо деньгами, либо 300 бонусами). Если настройка включена, то клиенты смогут частично оплачивать заказ (например, в заказе на триста рублей клиент сможет оплатить баллами 230 рублей и оставшиеся семьдесят — деньгами) (чтобы включить выберите Включить использование баллов и Возможность частичной оплаты заказа баллами).

![](https://howto.aqua-delivery.com/wp-content/uploads/2020/09/Screenshot_11.png)

Настройка **Возможность частичной оплаты** **для онлайн оплат** — если настройка выключена, то клиенты могут оплачивать заказ только целиком (например, заказ на триста рублей клиент сможет оплатить либо онлайн-оплатой, либо 300 бонусами). Если настройка включена, то клиенты смогут частично оплачивать заказ только при онлайн-оплате (чтобы включить выберите Включить использование баллов и Возможность частичной оплаты заказа баллами для онлайн оплат).
