В системе Aqua Delivery есть функционал для партнеров (на тарифе «Базовый» и выше). Для его подключения обратитесь в техническую поддержку [по ссылке](https://t.me/AquaDelivery_support_bot).

Чат-бот в Telegram будет отправлять следующие сообщения:

1. О создании заказа
2. Об отмене заказа
3. Оценка клиентом приложения (когда клиент оставил оценку в самом приложении, не в магазине приложений)
4. Новое сообщение в чате от клиента
5. Оценка клиентом заказа (при включенном авт. пуш-уведомлении «О доставленном заказе»)
6. Ежедневный отчет (в отчете находится информация по заказам, завершенным в данный день)
7. Отчет о пропущенных заказах.

Можно включить/выключить любой из перечисленных пунктов.
