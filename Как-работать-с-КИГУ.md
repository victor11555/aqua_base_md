# Как работать с КИГУ — База знаний Aqua Delivery

Логика работы с упаковкой товара следующая:

1. Настраиваем работу с упаковкой товара. Как настраивать работу с КИГУ.
2. Поступает заказ на упаковку товара, назначаем его на водителя.
3. При доставке заказа курьер сканирует только один код маркировки, который нанесен на упаковку товара, завершает заказ.
4. В заказе и в расчетной накладной в 1С будет указываться товар с единицей измерения.
