# Удаление аккаунта — База знаний Aqua Delivery

Чтобы удалить аккаунт в мобильном приложении, вам нужно выполнить несколько шагов:

Шаг 1. Перейдите в раздел «Профиль» в нижнем меню на главной странице.

![](https://howto.aqua-delivery.com/wp-content/uploads/2023/12/Рисунок1.jpg)

Шаг 2. В самом низу есть строка «Удалить аккаунт».

![](https://howto.aqua-delivery.com/wp-content/uploads/2023/12/photo_2025-03-12_15-37-40.jpg)

Шаг 3.  Выберите «Удалить аккаунт».

Шаг 4. Подтверждаем удаление аккаунта.

![](https://howto.aqua-delivery.com/wp-content/uploads/2023/12/photo_2025-03-12_15-41-24.jpg)

**Примечание:  
После удаления профиля его можно будет восстановить только через оператора.**
